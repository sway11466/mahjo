# キャラクター定義：まお（魔女）

> 世界・人物相関（町・占い館・師匠・系譜・呼称ルール）の正は [world.md](../world.md)。本書はまお個別の素性・ビジュアル・セリフを持つ。

## 1. 基本情報

- id：`mao`（レジストリ登録名・アセットパス `characters/mao/`）
- 表示名：まお（仮）
- 位置づけ：アプリ名 Mahjo（＝魔女）にちなむマスコット。既定キャラ。
- 好きな役：一気通貫（1〜9のまっすぐ一本＝「まっすぐ」な性格と掛かる）。手にあるとヒント／正解で専用セリフを優先（個性付け）。

---

## 2. ペルソナ

- 性格：真面目・まっすぐ・素直・優しい魔女見習い（タマゴ）。神秘的に構えたり怪しく見せたりせず、一緒に学んで寄り添う。
- 人称：一人称「わたし」。相手は `AppSettings.playerName` があればその名前、なければ「あなた」。
- 口調：やわらかい です・ます 基調。占い／魔法はほんのり添える程度で、命令でなく寄り添って励ます。段階ヒントは「ぼんやり→鮮明」へ。答え（役名・点数）は言わない。ミス時もプレッシャーをかけない（[product-concept](../../product-concept.md)）。セリフ本体は [character-mao-script.md](./character-mao-script.md)。

## 3. ビジュアル

外見コンセプト：

- 「洋風魔女の本質（神秘・占い）＋チャイナ服」の融合。道姑のような専門用語的モチーフは避け、誰が見ても“中華の魔女”と分かる親しみやすさを優先（神秘・占いは見た目の味付け。性格は §2 ペルソナ）。

- 帽子なし・ツインのお団子（紺色の髪・ぱっつん前髪）。**お団子の根元から、星＋短冊状の小さな御札（お札）の髪飾りを左右に短く下げる**（お団子に乗せず・耳飾りでもない＝髪から短く下がる）。前髪には飾りを付けない。両サイドに横髪（サイドロック）を垂らす。**肩に羽織る短いケープ（金縁・前開き・裏地無地＝ドレスと別布に見せる）**、丈長のチャイナドレス（深い紫／夜紺＋金縁・朱）で**詰め襟・胸元は閉じる**・サイドスリット、裾は膝下〜ふくらはぎ丈と長め。**靴はドレスと同系の深い紫**。瞳は琥珀（アンバー）。手には御札の束を扇状に広げて持つ（折り畳み扇ではなく平たい木の札。※これは master では持たせず派生ポーズで作る）。配色と星座・雲モチーフで「魔女」を読ませる、やわらかく親しみやすい雰囲気。肌は明るめ・血色控えめのフラット塗り。画風は**りんと共通のハウススタイル＝くっきり線＋フラットなセル塗り（ツヤ控えめ）**（backlog feature-11 で刷新）。

衣装の模様（確定。生成・表情差分で維持する正）：

- 地色＝深い紫／夜紺、縁＝金。
- 要素＝金の星（大小）／金の三日月／細い金の星座線（点を線で結ぶ）／朱赤の瑞雲（吉祥雲）。
- 密度は中程度、無地の紫を広く残す（盛りすぎない）。ドレス本体は各要素をバランスよく散らし裾まで連続、ケープは星座線が主体で控えめ（瑞雲は裾に少し）。
- 御札の札面の文字は赤系の装飾グリフ（判読不能でよい＝嘘字対策）。
- 線質：くっきりしたアニメ線＋フラットなセル塗り（ツヤ控えめ＝りんと共通のハウススタイル。眠い／もやっとした線にしない）。

モチーフ（用語の定義は [character-guide.md](../character-guide.md) §2「ビジュアル」）：

- 法具モチーフ＝符（お札）＝`motif.ritual: 'ofuda'`：三重の意味でテーマに噛み合う（魔女 まお が符を操り、その符は点数計算の符(fu)でもある）。回答選択マークに使う（resolver `selectionMark`→`Ofuda`）。ヒント時／点数モードで該当牌に符を貼る＝ハイライトの描画表現にも使える（演出はペルソナ側、対象データ＝`HighlightTarget` は中立。[data-model.md](../../design/data-model.md) §10）。
- 装飾モチーフ＝月・星＝`motif.decor: 'moon'`：背景に薄く展開する雰囲気レイヤ（三日月・星で衣装の月・星に呼応）。色は下記「差し色」（`accentColor`）が乗る。

テーマ色：

- まおの identity 主色＝世界観の紫（深い紫／夜紺）。暫定値 `themeColor: '#6a4fb0'`（濃さ・青/金寄りはブラウザで微調整）。
- 役割・使い方の汎用ルールは [character-guide.md](../character-guide.md) §2「テーマ色」。

差し色：

- まおの identity 従色＝金。`accentColor: '#f5c451'`（`--color-highlight` と同値で、金の刺繍・扇・髪飾りに呼応）。装飾モチーフ（月・星）の色に使う。
- 役割・使い方の汎用ルールは [character-guide.md](../character-guide.md) §2「差し色」。

使い魔（世界設定として全キャラが1体持つ。役割・仕組みは [character-guide.md](../character-guide.md) §2「ビジュアル」使い魔）：

- **ココ**（オス・猫）。中国語 猫＝「māo」がまおの名と重なる由来。まお⇄ココ は互いに呼び捨て。
- 性格：めんどくさがり・気だるげ・ボソッと毒舌、根は優しい隠れ世話焼き。しっかり者のまおと対。**人語を操れるが「喋るのもめんどう」で黙す**（＝アプリ内で声を出さない A方向の作中理由）。ヒントは答え・役名を言わず、だるそうに前足で示す → まおがひらめく。
- 見た目：自然な猫体型のアニメ猫（まお通常版に馴染む）。夜紺ベース＋琥珀の瞳（まおと揃える）、首輪／額に三日月・星・御札の小物＋金の差し色。顔は大きめのまん丸な琥珀の瞳でやわらかく親しみのある作り（**だるさは表情でなく脱力ポーズで出す**＝寝そべり・伸び・頬杖等は i2i 派生で）。線質・透過WebP は本体と同じ。
- アセット：単独キャンバスでおすわり全身を生成、表示寸は ui 調整（おすわり全身 ≒ まおの顔一個分が目安）。出し先＝ヒントボタン（この絵をクロップ）／トップ／キャラ選択。生成プロンプトは別途（本体に倣う）。

### 生成プロンプト

まお（＋使い魔ココ）の master・派生（道具付き立ち絵／バストアップ）・使い魔の**画像生成プロンプトは [character-mao-image-prompts.md](./character-mao-image-prompts.md) に分離**（本ファイルの肥大化回避）。本§の識別情報（外見コンセプト・模様・モチーフ・配色・使い魔設定）が正で、プロンプトはそれを参照して組む。作る画像セットと順序・一貫性ワークフローは [character-guide.md](../character-guide.md) §4。

## 4. セリフ

全セリフ（場面別・ヒント・解説・誤答）は [character-mao-script.md](./character-mao-script.md)。共通規則は [character-guide.md](../character-guide.md) §2「セリフ」、口調と「答え（役名・確定値）を言わない」原則は §2 ペルソナ。

- 個性フック：好きな役＝一気通貫（§1）。手にあるとき `correct`／`explaining` は専用セリフを優先（解説に「…わたしの好きな役なんです」等）。ヒント中は役名を言わず嬉しさだけ滲ませる。
