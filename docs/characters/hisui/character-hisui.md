# キャラクター定義：ひすい（魔女・姉弟子格）

> 世界・人物相関（町・占い館・師匠・系譜・呼称ルール）の正は [world.md](../world.md)。本書はひすい個別の素性・ビジュアル・セリフを持つ。
> ひすいの系譜上の位置（ロウシ直弟子＝師匠世代の末弟子／まお・りんの師叔格だが「ひすい姉さん」と慕われる両世代の橋渡し）・弟子しんとの関係の正は [world.md](../world.md) §5。

## 1. 基本情報

- id：`hisui`（レジストリ登録名・アセットパス `characters/hisui/`）
- 表示名：ひすい
- 位置づけ：**姉弟子格**。大師匠 ロウシ の**直弟子**＝こはく・さんご と同世代（格は師匠たちと同列）だが、その中で**最も若い末弟子**。まお・りんから見ると一世代上＝師叔格だが、歳が近く（師匠たちより若く、まお・りんより年上）、親しみで「**ひすい姉さん**」と慕われ、師匠たちからは妹弟子として可愛がられる＝**両世代の橋渡し**。**知的・聡明**で、魅力は色気でなく知性（ゴージャス・動のさんごとは静↔動・シック↔ゴージャスで棲み分ける）。**しんの師**。プレイヤーには落ち着いて寄り添う、頼れる姉（咎めない＝[product-concept](../../product-concept.md) §3）。
- 好きな役：**緑一色（リューイーソー）**（索子の緑牌と發だけで作る「**緑だけの手**」＝**翠**＝彼女の色そのもの。清廉・静けさにも通じる。色が直に一致する最強の identity フック）。手にあるとヒント／正解で専用セリフを優先（個性付け）。
  - ※役満は現状の生成では出題されない（役満シードは parking lot＝[scoring-rules.md](../../spec/scoring-rules.md) §1.2）ため、フックは役満生成の対応まで実発火しない（さんごの大三元と同様）。identity の良さを優先した確信犯。

---

## 2. ペルソナ

- 性格：**知的・聡明・温和**。落ち着いていて余裕があり、急かさず・偉ぶらず、ゆったり導く姉。賑やかな まお／小悪魔の りん に対し、**静かな聡明さ**で立つ。弟子しんを見守る面倒見の良さ。色気でなく知性で魅せる（ゴージャス・動のさんごに対し、シック・静で別路線）。プレイヤーを咎めたり急かしたりしない（[product-concept](../../product-concept.md) §3「プレッシャーをかけない」を死守）。
- 人称：一人称「**わたし**」。相手は `AppSettings.playerName` があればその名前、なければ「あなた」（穏やかな姉の距離感）。
- 口調：丁寧めで穏やかな**姉口調**。「〜ね」「〜でしょう？」「ゆっくりでいいのよ」と落ち着いて導く。知的だが説教くさくしない・上から言わない。段階ヒントは「ぼんやり→鮮明」へ。答え（役名・点数）は言わない。ミス時はからかわず優しく寄り添う（[character-guide.md](../character-guide.md) §2「セリフ」）。セリフ本体は [character-hisui-script.md](./character-hisui-script.md)。
- reactions（場面→表情の上書き。既定マップは [session.md](../../spec/session.md) §4）：**上書きなし＝`reactions: {}`**。neutral 自体が穏やかな微笑なので既定どおりで成立する（greeting=neutral／dealing=thinking／hinting=insight／explaining=smile／correct=happy／wrong=troubled／finished=happy）。

## 3. ビジュアル

外見コンセプト：

- 「中華の占い師（魔女は表に出さない＝[world.md](../world.md) §7）＋聡明で落ち着いた姉」。まお・りんが10代の見習いなのに対し、ひすいは**少し年上の落ち着いた若い女性（20代前半）**。背は高め・所作は静かで品がある。誰が見ても「頼れる年上の占い師・ひすい姉さん」と分かる、知的で穏やかな佇まい。色気でなく**聡明さ・余裕**で魅せる（佇まいはシック・静＝ゴージャス・動のさんごと対）。
- 帽子なし・**ロングのハーフアップ**（上半分まとめ・残りは長く流す。まおのお団子・りんのツインテールと差別化）、ぱっつん前髪、長いサイドの後れ毛。**艶のある翠みの暗髪**。瞳は**翠**（まおの琥珀・りんの紅と差別化）。**穏やかで聡明な閉じ口の微笑**（きつくしない・色っぽくしない）。**翡翠の小さな簪＋ごく小さな勾玉の下がり**（大ぶりにしない＝制作時に小型化を確定）。画風は**まお・りんと共通のハウススタイル＝くっきり線＋セル塗り（ツヤ控えめ）**。

衣装の模様（確定。生成・表情差分で維持する正）：

- 地色＝**翡翠グリーン（澄んだ翡翠／エメラルド翡翠。青緑・薄ミント・黒緑にしない）**、縁・差し色＝**真珠白**。
- 形＝**床丈のロングチャイナ（一枚もの・足首丈・サイドスリット）・襟詰め（胸元クローズ＝控えめを死守）**、フラット靴はドレス同系（翡翠）。羽織りは**両腕に掛けて床まで流れる薄手・透ける真珠白の披帛（ショール）**（まお＝短ケープ／りん＝長ケープに対する差別化。ケープではない）。
- **胸元＝勾玉形のチャイナボタン留め具（真珠白の盤扣＋翠の勾玉トグル）**＝法具モチーフ勾玉を胸元に乗せる identity 要素。
- 要素＝**真珠白の水紋（同心円アーチ＝青海波状）**／**勾玉（翠の差し色）**。雲（まお）・蝶（りん）・焔（しん）は使わない。
- ゾーン配置（非対称・余白広め）＝裾：水紋の帯／腰（片側）：水紋の小クラスタ＋勾玉／胸元：勾玉の留め具。**披帛（薄布）には水紋を乗せない**（左右対称フレームだと両肩に出て煩い＝バストアップ crop では留め具のみ）。盛りすぎない（無地を広く残す＝聡明で端正）。
- 線質：くっきりしたアニメ線＋セル塗り（眠い／もやっとした線にしない）。

モチーフ（用語の定義は [character-guide.md](../character-guide.md) §2「ビジュアル」）：

- 法具モチーフ＝**勾玉（まがたま）**＝`motif.ritual: 'magatama'`：占術の依代。回答選択マークに使う（resolver `selectionMark`→`Magatama`〔予定〕）。選んだ4択に**勾玉を翳す→翠に灯る**所作。ヒント時／点数モードで該当牌に勾玉を添える＝ハイライトの描画表現にも使える（演出はペルソナ側、対象データ＝`HighlightTarget` は中立。[data-model.md](../../design/data-model.md) §10）。皮の SVG は `ui/character/items/Magatama.tsx`（予定）。
- 装飾モチーフ＝**水紋（みずもん）**＝`motif.decor: 'mizumon'`：背景に静かに広がる同心円の波紋（青海波状）。**水＝知恵・清廉**（弁財天の白蛇）に通じ、まおの「月・星（天）」／りんの「蝶（妖）」／しんの「焔（火）」に対して**水・静**で対照をなす。色は下記「差し色」（`accentColor`＝真珠白）が乗る。点でなく**形**なので他キャラの装飾と被らない。キーは開いた文字列なので型変更は不要、解決は ui の resolver（`ui/character/decor/`）。

テーマ色：

- ひすいの identity 主色＝**翠（翡翠グリーン）**。暫定値 `themeColor: '#3f9e7c'`（澄んだ翡翠。青緑〔teal〕に寄せず緑寄り・薄くしすぎない。ブラウザで微調整）。
  - **他キャラとの差別化（色）：** まお＝紫／りん＝赤紫／さんご＝紅／しん＝黒 に対し、ひすいは**緑系で唯一**＝並べて即・別人。落ち着いた中明度の翡翠で「知的・清廉・静」を出す。
- 役割・使い方の汎用ルールは [character-guide.md](../character-guide.md) §2「テーマ色」。状況では変えない（誤答で色を変える等はしない＝プレッシャーをかけない）。

差し色：

- ひすいの identity 従色＝**真珠白**。暫定値 `accentColor: '#eef0e8'`（披帛・水紋・縁の真珠白に呼応。わずかに温かみのある乳白）。装飾モチーフ（水紋）の色に使う。まおの金（暖）・りんの銀（寒）・しんの緋（火）に対し、ひすいは**真珠白（清）**。
- 役割・使い方の汎用ルールは [character-guide.md](../character-guide.md) §2「差し色」。

使い魔（世界設定として全キャラが1体持つ。役割・仕組みは [character-guide.md](../character-guide.md) §2「ビジュアル」使い魔・[decisions](../../decisions.md) 2026-06-10）：

- **白蛇**（弁財天の神使＝**知恵・清廉**）。聡明なひすいの相棒で、声を出さずヒントをそっと示す → ひすいがひらめいて言葉にする（A方向）。
- 質感：白〜真珠白の体に**翠の瞳**、所により鱗へ翡翠の差し。理知的で清らか、怖くしすぎない（神使の気品＋親しみのある相棒）。ひすい本体に配色・画風を寄せる。
- 役割の含み：弁財天の白蛇＝知恵と水の神使＝ひすいの「聡明さ・水紋」と縫合（弟子しんのカラス〔母の血＝朱雀の翳り〕とは別系統。[world.md](../world.md) §5）。
- 見た目の詳細は本体の絵が固まってから詰める。当面は設定のみで UI 未登場。出し先＝ヒントボタン（この絵をクロップ）／トップ／キャラ選択。

## 4. セリフ

全セリフ（場面別・ヒント・解説・誤答）は [character-hisui-script.md](./character-hisui-script.md)（未作成＝今後）。共通規則は [character-guide.md](../character-guide.md) §2「セリフ」、口調と「答え（役名・確定値）を言わない」原則は §2 ペルソナ。穏やかな姉口調・必須場面（greeting/dealing/correct/wrong/finished）は各最低2本（飽き対策）。

- 個性フック：好きな役＝緑一色（§1）。手にあるとき `correct`／`explaining` は専用セリフを優先（解説に「緑だけで揃った……わたしの色でしょう？」等、答え＝役名は言わずに匂わせる）。ヒント中は役名を言わず、嬉しさだけ穏やかに滲ませる。
- 姉味：`greeting`／`finished` に、まお・りんを見守る年上の落ち着き・両世代の橋渡しらしさを軽く滲ませてよい（プレイヤーへの煽りにはしない）。
