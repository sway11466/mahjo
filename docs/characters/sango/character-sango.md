# キャラクター定義：さんご（魔女・りんの師匠）

> 世界・人物相関の正は [world.md](../world.md)（§4 占術院・§5 系譜）。本書は さんご個別の素性・ペルソナを持つ。**構想中＝ペルソナ先行**で、ビジュアル・セリフ・画像は TBD。

## 1. 基本情報（最小・正は world.md §4/§5）

- 位置づけ：りんの師匠＝りんの館（赤い館）の館主。大人の女性。ロウシの直弟子＝師匠世代。
- 出自：まおの町の財界の有力者の娘。ただのボンボンではなく努力家。家業は順調で、都会の貴族にも引けを取らない（むしろ上回る）富。都会にも家を持つが、本人はこの町を好む。
- 格：占い師としての格は家の力でなく実力で勝ち取った＝努力で館主に成り上がった。世俗の格（財界）と占い師の格は別物（world.md §4）。
- 色＝珊瑚紅。使い魔＝狐仙（九尾）の艶やかさ・大人の色気を映す（詳細・ビジュアルは TBD）。
- 好きな役：大三元（ダイサンゲン）（三元牌の「中（紅）」＝珊瑚紅に呼応／絢爛豪華＝繁盛するエレガントな館・館主の大物感／役満の格＝努力で勝ち取った実力者の貫禄・狐仙（九尾）の艶やかさ）。手にあるとヒント／正解で専用セリフを優先（個性付け）。
- `unlock` 等：TBD。

## 2. ペルソナ

- 性格：きつめでしっかり者の実力派・努力家。弟子にも自分にも厳しいが、根は面倒見がよく情に厚い。大人の余裕と落ち着き・華やかな大人の色気をまとう。出自の富に驕らず実力主義（成り上がりの矜持）。
- 類型＝ツンデレ：好意・優しさを素直に出せず、つい きつく当たる／そっぽを向く。照れると否定する（「べ、別に」）。デレが漏れるギャップが魅力。
  - 対比：弟子りん＝小悪魔・からかい（好意を“見せて”楽しむ）↔ さんご＝ツンデレ（好意を“隠して”否定する）。師弟で攻め／守りが裏返る（world.md §5）。
- こはく（まおの館の館主）へ：表向き対立してきつく当たるが、根は好いている（“走らない”こはくへの苛立ちと好意がないまぜ＝world.md §5）。
- 弟子 りん へ：実力を認め合う師弟。だが互いに本心を出さないため、傍目には不仲・心配に見える（実際は信頼が下地＝world.md §5）。
- 人称・口調：TBD（姉御肌・きつめ想定）。プレイヤー向けセリフ（`Persona` プール）は support キャラとして実装する段で別途。
