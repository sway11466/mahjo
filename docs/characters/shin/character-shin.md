# キャラクター定義：しん（魔女・ひすいの弟子）

> 世界・人物相関（町・占い館・師匠・系譜・呼称ルール）の正は [world.md](../world.md)。本書はしん個別の素性・ビジュアル・セリフを持つ。
> しんの**出自（裏設定＝堕ちた占い師の子・母の朱雀 等）の正は [world.md](../world.md) §5**。裏設定は本書・world.md 本文にのみ記し、相関図には出さない（world.md §5 の運用ルール）。

## 1. 基本情報

- id：`shin`（レジストリ登録名・アセットパス `characters/shin/`）
- 表示名：しん（仮）
- 位置づけ：**ひすいの弟子**（ロウシ→ひすい→しん の孫弟子）。まお・りんと同世代だが、**どちらの占い館にも属さない中立**の少年。クール・寡黙な観察者。**出自は謎**（裏の正は [world.md](../world.md) §5）。プレイヤーには静かに寄り添う（咎めない）。
- 好きな役：三暗刻（サンアンコー）（**暗刻＝隠れた・内に秘めた刻子**＝しんの秘めた出自・寡黙と掛かる。ロンで明刻化して崩れる＝「晒すと消える」繊細さも）。手にあるとヒント／正解で専用セリフを優先（個性付け）。

---

## 2. ペルソナ

- 性格：クール・寡黙・観察眼が鋭い。多くを語らず要点だけを置く。母の血（闇＝邪な占術）を律する**克己の静けさ**で、冷たく突き放すのではなく*静かに見守る*（[product-concept](../../product-concept.md) §3「プレッシャーをかけない」を死守）。まお・りんの賑やかさに対し、静の対比を立てる。
- 人称：一人称「**俺**」。相手は `AppSettings.playerName` があればその名前、なければ「お前」（突き放さない範囲の素っ気なさ。煽り・命令にはしない）。
- 口調：言葉数が少なく淡々・短い断定・タメ口。「…」の間を取り、たまに核心を突く一言。段階ヒントは「ぼんやり→鮮明」へ。答え（役名・点数）は言わない。ミス時も咎めず静かに寄り添う（[character-guide.md](../character-guide.md) §2「セリフ」）。セリフ本体は [character-shin-script.md](./character-shin-script.md)。
- reactions（場面→表情の上書き。既定マップは [session.md](../../spec/session.md) §4）：
  - `correct`：happy → **smile**（控えめに認める。大喜びはしない＝クール）。
  - その他（greeting=neutral／dealing=thinking／hinting=insight／explaining=smile／wrong=troubled／finished=happy）は既定どおり（finished の喜びも淡め）。

## 3. ビジュアル

外見コンセプト：

- 「中華の占い師（魔女は表に出さない＝[world.md](../world.md) §7）＋陰のあるクール美少年」。ひすいの弟子＝端正だが影がある。**黒×緋**で、まお（まっすぐ）・りん（妖しい得意げ）に対して*静かで鋭い*を立てる。誰が見ても「もう一人の、影のある少年」と分かる佇まい。
- ビジュアルの参考イメージ＝アニメ的な*黒のとがった毛束・クールな目つき・ダークな衣装・鮮烈な発光*の少年像（※特定の版権キャラを直接模倣しない。特徴を記述して寄せる＝§生成プロンプト・[character-guide.md](../character-guide.md) §4 嘘字/版権の注意）。

- 帽子なし・**黒髪（つや黒）＋動きのあるとがった毛束**、顔まわりに落ちる**長め前髪**（暗すぎない＝片目を完全には隠さない）。**緋のインナーカラー一筋**（差し色＝母の血＝朱雀の火の気配）。**瞳は緋（朱）**＝黒髪に緋眼で出自を仄めかす（差し色と縫合）。**黒基調の道服／チャイナ**（襟詰め・やや丈長・動きやすい軽装）、**緋の裏地・縁**。配色（黒×緋）と卦札（法具）・焔（装飾）のモチーフで「影のある占いの子」を読ませる。画風は**まお・りんと共通のハウススタイル＝くっきり線＋フラットなセル塗り（ツヤ控えめ）**。

衣装の模様（確定。生成・表情差分で維持する正）：

- 地色＝黒、差し色・縁＝緋。
- 要素＝**焔（緋の炎の舌・内くり抜き）**／**カラスの羽根**／**卦の爻線（陰陽の横線）**。雲（まお）・蝶（りん）は使わない。
- 密度は**低め・非対称・余白を広く残す**（クールでミニマル＝賑やかな まお/りん と対比）。緋は面で使わず**縁・裏地・一筋で点的に**効かせる（黒が主役）。
- 卦札の札面・羽根の意匠は緋の装飾グリフ（判読不能でよい＝嘘字対策）。
- 線質：くっきりしたアニメ線＋フラットなセル塗り（眠い／もやっとした線にしない）。

モチーフ（用語の定義は [character-guide.md](../character-guide.md) §2「ビジュアル」）：

- 法具モチーフ＝**卦札（けふだ）**：卦（八卦／易の爻線）を記した占いカード。回答選択マークに使う（resolver `selectionMark`→`Kefuda`〔予定〕）。選んだ4択に**札を伏せる→めくる→緋に灯る**所作。まおの符（縦長の呪符）と差別化＝*角丸のカード*・卦の爻線が主役・裏面は緋縁／カラス羽根の意匠。皮の SVG は `ui/character/items/Kefuda.tsx`（予定）。
- 装飾モチーフ＝**焔（ほのお）**：揺れる炎の舌（内くり抜きのネガティブスペース）を緋で散らす雰囲気レイヤ。*抑えた炎*＝朱雀の血の気配（出自は出さず気配だけ）。色は下記「差し色」（`accentColor`）が乗る。点でなく**形**なので、まおの星きらめきと被らない。実SVGは作者が描く予定（仮素材で先行配線可）。

テーマ色：

- しんの identity 主色＝**黒（墨黒に僅かな深み）**。暫定値 `themeColor: '#1c1b24'`（純黒だとフラットなので僅かに青/紫みの深い黒。背景グローは影のように沈む＝陰のある少年に整合）。
- 役割・使い方の汎用ルールは [character-guide.md](../character-guide.md) §2「テーマ色」。状況では変えない（誤答で色を変える等はしない＝プレッシャーをかけない）。

差し色：

- しんの identity 従色＝**緋（鮮烈な赤）**。暫定値 `accentColor: '#e0392c'`（瞳・インナーカラー・縁・卦札の発光に呼応）。装飾モチーフ（焔）の色に使う。まおの金（暖）・りんの銀（寒）に対し、しんは**緋（火種）**＝抑えた炎。
- 役割・使い方の汎用ルールは [character-guide.md](../character-guide.md) §2「差し色」。

使い魔（世界設定として全キャラが1体持つ。役割・仕組みは [character-guide.md](../character-guide.md) §2「ビジュアル」使い魔・[decisions](../../decisions.md) 2026-06-10）：

- **カラス**（黒・高知能）。寡黙なしんの相棒で、声を出さずヒントをそっと示す → しんがひらめいて短く言葉にする（A方向）。
- 質感：黒一色の理知的なカラス。鋭い目だが不吉に振りすぎない（陰はあるが相棒）。緋の差し（目・足環 等）でしんと揃える。
- 役割の含み：黒いカラスは*母の朱雀（＝太陽の三足烏とも通じる火の鳥）の翳った写し*＝師ひすいの白蛇ではなく*母の血*を密かに引く（裏設定。[world.md](../world.md) §5）。アプリ内では出自を語らず、見た目の気配に留める。
- 見た目の詳細は本体の絵が固まってから詰める。当面は設定のみで UI 未登場。出し先＝ヒントボタン（この絵をクロップ）／トップ／キャラ選択。

### 生成プロンプト

しん（＋使い魔カラス）の master・派生（道具付き立ち絵／バストアップ）・使い魔の**画像生成プロンプトは [character-shin-image-prompts.md](./character-shin-image-prompts.md) に分離**（本ファイルの肥大化回避。未作成＝今後）。本§の識別情報（外見コンセプト・模様・モチーフ・配色・使い魔設定）が正で、プロンプトはそれを参照して組む。参考イメージ（黒のとがった髪・クールな目・ダーク道服・緋の発光）は*特徴を記述して寄せる*＝版権キャラの直接模倣は避ける（[character-guide.md](../character-guide.md) §4）。作る画像セットと順序・一貫性ワークフローは [character-guide.md](../character-guide.md) §4。
