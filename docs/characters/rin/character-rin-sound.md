# キャラクター音楽（BGM）：りん

> りん個別の素性・ビジュアル・セリフは [character-rin.md](./character-rin.md)。本書はりんの BGM（主旋律・即興音）を持つ。

りんの BGM。コード生成方式（Web Audio 合成・音源ファイルなし）の二層構成＝主旋律（キャラごとの決まった旋律・度数記法）＋即興音（主旋律の合間を飾る毎回ちがう音・セグメント配列）。記法の編集・試聴は開発ツール [tools/melody-authoring](../../../tools/melody-authoring/) で行う（各設定値の意味・取り得る値もツールに集約）。

> BGM の実現方式・度数記法・即興音のセグメント配列の正は [sound.md](../../design/sound.md)「BGM の実現方式」。判断の記録は [ADR-0003](../../adr/ADR-0003-bgm-code-generation.md)。本書はりんの曲データを持つ。

## 主旋律

方向性＝小悪魔・軽やか＋上流の艶。まおの「まっすぐで元気（宮調・撥弦）」に対し、りんは「対でころころ鳴る（徵調・揚琴）」で対比する。フレーズを「同じ音を2つ重ねる」対の動機（`3 3` `5 5` `1' 1'`）で組み、好きな役＝七対子（対の繰り返し）と名前⇄鈴（りんりん）を旋律の骨にする。中盤（サビ）は高音を飛び回る蝶のフルッター、終わりは着地せず上げて「ふふっ」と締める（ループでまた頭へ＝小悪魔のウインク）。暗く沈まないよう翳りの調（角・羽）は避け、明るく開けた徵調に置く。音色は揚琴（ヤンチン＝叩いた金属弦）で、撥弦（琴）の爪弾きと分ける「叩いた立ち上がり」を持たせる。

```
instrument: yangqin      // 揚琴（ヤンチン）＝叩いた金属弦
tonic:      D5           // 主音（まお C5 より少し高く、鈴の繊細さ）
mode:       徵           // 明るく開けた調（翳りの角・羽は避ける）。音程 0,2,5,7,9（主音からの半音）
tempo:      110          // BPM
brightness: 5200         // Hz（明るさ＝ローパス）
sustain:    0.5          // 余韻（揚琴の伸び）
melody:     3 3 5 5 3 2 3:2 5 5 3 3 2 1 2:2 5 5 1' 1' 5 3 5:2 1' 1' 5 5 3 5 3:2 5 1' 2' 1' 5 1' 2':2 3' 2' 1' 2' 1' 5 3:2 3 3 2 2 3 5 3:2 2 2 3 3 5 3 5:2
```

度数記法：度数 `1`〜`5`（音階の低い方から・主音=1）／オクターブ `'` 上 `,` 下／長さ `:拍`（既定1）／休符 `0`／空白区切り。モード（調）は同じ度数でも実音程が変わる（宮＝明るい／羽＝翳り 等）。

## 即興音

即興音（主旋律の合間を飾る毎回ちがう音）。セグメント配列＝`{ start, length, instrument, density, octave, offset, volume, brightness, sustain }`（`start`＝開始小節・`start+length` で配置、同じ `start` に重ねると声部が重なる）。前半は控えめな撥弦（琴）の裏拍きらめき（density 1）、後半（サビ含む）は撥弦を密（density 2）にしつつ、揚琴の連弾レイヤー（density 4・表拍）を重ねて厚みを出す（サビの旋律が単音の走句で対の重ねが抜けるぶんを下から補う）。各項目の意味・取り得る値は開発ツール（冒頭リンク）に集約。

```
即興音 = [
  { start: 0, length: 8, instrument: pluck,   density: 1, octave: 1, offset: 0.5, volume: 0.30, brightness: 5000, sustain: 0.5 },
  { start: 8, length: 8, instrument: pluck,   density: 2, octave: 1, offset: 0.5, volume: 0.30, brightness: 5000, sustain: 0.5 },
  { start: 8, length: 8, instrument: yangqin, density: 4, octave: 1, offset: 0,   volume: 0.26, brightness: 5200, sustain: 0.4 },
]
```
