# りん セリフ script（全量・仮）

りんの全セリフの正本。**場面別セリフ（§1）＋着目ポイント別ヒントセリフ（§2）＋成立役の解説セリフ（§3）＋誤答の諭しセリフ（§4）**を網羅する。ヒント・解説は [hint-base.md](../../spec/hint-base.md) の役/ドラ キーをカバーする（突き合わせ＝バリデーション対象）。口調・性格はキャラ定義 [character-rin.md](./character-rin.md) §2 ペルソナに従う（タメ口・小悪魔・からかい上手／ミス時は寄り添う）。§1 の必須場面（あいさつ／出題／正解／ミス／終了）は確定。ヒントセリフ（§2）と任意場面は仮で順次。§3 解説は通常役＋ドラを実作、役満・レア役は順次（parking lot）。

---

## 1. 場面別セリフ

場面ごとに表情（`Expression`）＋セリフ。複数からランダム表示（飽き対策）。場面の正準リスト・必須度は [session.md](../../spec/session.md) §4 場面（リアクション）の正準、セリフ側の規則は [character-guide.md](../character-guide.md) §2「セリフ」に従う。りんは小悪魔なので一部の表情を既定マップから上書きする（`reactions: { correct: 'mischievous', greeting: 'smug' }`）。それ以外の場面（dealing→neutral／wrong→troubled／finished→happy）は既定どおりなので表情は併記しない。

### あいさつ（開始）（greeting → smug｜必須）
セッション開始の1回。まおへの一方的な対抗心を軽く滲ませてよい（相手はまお／プレイヤーは煽らない）。
- ふふっ、来たね。あたしの方が上手に視てあげる。
- まおより楽しませてあげる。覚悟しておいてね？
- お待ちかね。さ、鈴を鳴らそうか。

### 出題（各問）（dealing → neutral｜必須）
毎問の出題中（回答前）。答えは言わない。
- この手……さて、キミには何が見える？
- 落ち着いて。あたしが見てるから大丈夫。
- ふぅん、なかなか面白い手だね。
- ひとつずつ、ね。慌てない慌てない。

### 正解時（correct → mischievous｜必須）
得意げに、でも嬉しそうに。
- ふふっ、お見通しだったでしょ？
- 正解。やるじゃない、見直しちゃった。
- そうそう、それでこそ。
- あたしの見込んだ通り。いい目してるね。
- 鈴の音が聞こえた？　大当たり。

### ミス時（wrong → troubled｜必須）
小悪魔でも、ここはからかわず寄り添う。方向づけ（どこを間違えたか）は §4 誤答の諭しが担い、誤答時は「ここ（寄り添い）＋ §4（諭し）」を結合して出す。
- あー、惜しい。責めてないよ？
- ん、ちょっとずれたね。だいじょうぶ。
- 惜しかった。鈴も少し笑ってる、優しくね。
- そこは、あたしと一緒に見直そっか。

### 終了（結果）（finished → happy｜必須）
全8問を終えた結果画面。点数は画面側が出すので、お祝い・ねぎらいだけ。まお味を軽く添えてよい。
- おつかれさま。ね、あたしと組んだ方が楽しいでしょ？
- ぜんぶ視てきたね。今日はよくがんばった。
- ここまで来たキミに、鈴ひとつ。また鳴らしにおいで。

## 3. 解説セリフ（成立役の説明）

回答後の解説シーンで、りんが成立役を **1つずつ** 説明する文言（[screens.md](../../design/screens.md) §3 解説シーン）。ヒント（§2）と違い回答後なので **役名を出してよい**。1キー＝1役の説明（1〜2文）。該当牌などは画面側がハイライトするので、文は役の概念＋ひとことにとどめ、特定の牌列に依存させない（「ここ」程度の軽い指示はOK）。キーはヒントと同じ語彙（`yaku:<id>` ／ `dora`・`aka-dora`・`ura-dora`）。配列でなく1文（段は持たない）。口調は小悪魔（タメ口・得意げ）だが、ここでも煽らない。

対象は通常役（[scoring-rules.md](../../spec/scoring-rules.md) §1.1）＋ドラ。役満（§1.2）・レア役（§1.3）は順次（parking lot で生成されるまで仮）。

### 役モード（`yaku:*`）

- `yaku:riichi`（リーチ）：リーチだね。門前（メンゼン）のまま聴牌（テンパイ）して宣言する――それだけで一翻。潔くていいでしょ。
- `yaku:double-riichi`（ダブルリーチ）：最初の巡目でのリーチ、「ダブルリーチ」。早い決断、嫌いじゃないな。
- `yaku:ippatsu`（一発）：リーチのすぐあと、一巡以内であがれた「一発（イッパツ）」。流れに乗ったね。
- `yaku:menzen-tsumo`（門前清自摸和）：門前（メンゼン）のまま自分で引いた「門前清自摸和（メンゼンツモ）」。このツモの一枚だよ。
- `yaku:pinfu`（平和）：順子（ジュンツ）だけ・役のない雀頭（ジャントウ）・両面（リャンメン）待ち……符（フ）のつかない綺麗な形、「平和（ピンフ）」。素直だね。
- `yaku:tanyao`（断幺九）：端の1・9も字牌（ジハイ）も使わない「断幺九（タンヤオ）」。2〜8だけ、すっきり。
- `yaku:iipeikou`（一盃口）：同じ並びの順子（ジュンツ）が二組、「一盃口（イーペーコー）」。ほら、そっくりでしょ？
- `yaku:yakuhai-haku`（役牌 白）：三元牌（サンゲンパイ）の白をそろえた役牌（ヤクハイ）。白は揃えるだけで一翻、お得だよ。
- `yaku:yakuhai-hatsu`（役牌 發）：發をそろえた役牌（ヤクハイ）。緑のこの組だね。
- `yaku:yakuhai-chun`（役牌 中）：中をそろえた役牌（ヤクハイ）。赤いこの組。
- `yaku:yakuhai-round`（場風）：場の風の牌をそろえた役牌（ヤクハイ）。今の場と同じ風、ね。
- `yaku:yakuhai-seat`（自風）：キミの風の牌をそろえた役牌（ヤクハイ）。自分の風と同じ、ここ。
- `yaku:sanshoku-doujun`（三色同順）：同じ並びを三つの色でそろえた「三色同順（サンショクドウジュン）」。色がそろうと気持ちいいよね。
- `yaku:sanshoku-doukou`（三色同刻）：同じ数の刻子を三色、「三色同刻（サンショクドウコー）」。なかなか珍しい形、よく作ったね。
- `yaku:ittsuu`（一気通貫）：ひとつの色で1から9までまっすぐ一本、「一気通貫（イッキツウカン）」。……まお好みの役だけど、悪くないね。
- `yaku:chanta`（混全帯幺九）：どの組にも端か字牌（ジハイ）が入る「混全帯幺九（チャンタ）」。全部に么九（ヤオチュー）、見える？
- `yaku:junchan`（純全帯幺九）：端の1・9だけで字牌（ジハイ）を使わない「純全帯幺九（ジュンチャン）」。チャンタより一段上、やるね。
- `yaku:chiitoitsu`（七対子）：面子（メンツ）を作らず、対子（トイツ）を七つそろえた「七対子（チートイツ）」。全部が対……あたしの鈴みたいでしょ？　好きな役なんだ。
- `yaku:toitoi`（対々和）：順子（ジュンツ）を使わず刻子（コーツ）だけでそろえた「対々和（トイトイ）」。組ばかりの力強い形だね。
- `yaku:sanankou`（三暗刻）：自分でそろえた暗刻（アンコー）が三つ、「三暗刻（サンアンコー）」。ロンじゃない、ここの組たち。
- `yaku:sankantsu`（三槓子）：槓（カン）を三つ作った「三槓子（サンカンツ）」。槓、頑張ったね。
- `yaku:honroutou`（混老頭）：1・9と字牌（ジハイ）だけの「混老頭（ホンロウトウ）」。真ん中の牌がひとつも無いの、気づいた？
- `yaku:shousangen`（小三元）：白・發・中のうち二つが刻子（コーツ）、一つが雀頭（ジャントウ）の「小三元（ショウサンゲン）」。三元牌（サンゲンパイ）、集めたね。
- `yaku:honitsu`（混一色）：数牌（スーパイ）を一色だけ＋字牌（ジハイ）の「混一色（ホンイツ）」。色がそろうと見ごたえあるでしょ。
- `yaku:ryanpeikou`（二盃口）：一盃口（イーペーコー）が二組ぶん、「二盃口（リャンペーコー）」。門前（メンゼン）でしか作れない、贅沢な形だよ。
- `yaku:chinitsu`（清一色）：字牌（ジハイ）も混ぜず、一色だけでそろえた「清一色（チンイツ）」。お見事、認めるよ。
- `yaku:haitei`（海底摸月）：最後のツモであがる「海底摸月（ハイテイ）」。ぎりぎりの一枚、引き寄せたね。
- `yaku:houtei`（河底撈魚）：場の最後の一枚でロンする「河底撈魚（ホウテイ）」。最後まで見てた成果。
- `yaku:rinshan`（嶺上開花）：槓（カン）のあとの嶺上牌（リンシャンパイ）であがる「嶺上開花（リンシャンカイホー）」。粋な巡り合わせ。
- `yaku:chankan`（槍槓）：相手の加槓（カカン）に重ねてあがる「槍槓（チャンカン）」。よく見てたね、お見事。

### ドラ

- `dora`（ドラ）：ドラだよ。ドラ表示の次の牌が、この手にあったね。一枚ごとに一翻、お得でしょ。
- `aka-dora`（赤ドラ）：赤ドラ。赤い5が混じってた。これも一枚で一翻。
- `ura-dora`（裏ドラ）：裏ドラ。リーチであがったご褒美、裏も乗ったね。

### 役満・レア役（順次）
`yaku:kokushi` ほか役満（[scoring-rules.md](../../spec/scoring-rules.md) §1.2）・レア役（§1.3）は、生成される parking lot で実作（仮・未）。

## 4. 誤答の諭しセリフ（MistakeKind）

クイズで誤答したとき、選んだ誤答の `MistakeKind` に応じてりんがそっと諭す文言（[screens.md](../../design/screens.md) §3、中立の基準は [hint-base.md](../../spec/hint-base.md)「誤答の諭し素」）。全 `MistakeKind` を網羅（型 `Record<MistakeKind, string>` で強制）。**答え（正解値）は言わない**。誤答セリフは「§1 ミス時（寄り添い・rng）＋ ここ（具体の諭し）」を結合して出す（寄り添い＋具体）。種類が取れないときは §1 の寄り添いのみ。小悪魔でも、ここはからかわず手引きする。

- `dealer-swap`（親子取り違え）：もしかして、親と子を取り違えてない？ 自分の風が東かどうかで点数が変わるよ。
- `dora-miss`（ドラ見落とし）：ドラ、見落としてない？ 赤い5や、リーチなら裏ドラも数えてみよ。
- `tsumo-ron-swap`（ツモ/ロン取り違え）：ツモとロン、取り違えてない？ 符（フ）や払いが少し変わるんだ。
- `fu-miscount`（符の数え違い）：符（フ）の数え、どこかで食い違ったかも。待ちや刻子（コーツ）、雀頭（ジャントウ）をていねいに見てみよ。
- `han-miscount`（翻の数え違い）：翻（ハン）の数え、ひとつ抜けたかもね。役をもう一度、かぞえ直してみよ。
