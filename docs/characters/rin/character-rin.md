# キャラクター定義：りん（魔女・ライバル）

> 世界・人物相関（町・占い館・師匠・系譜・呼称ルール）の正は [world.md](../world.md)。本書はりん個別の素性・ビジュアル・セリフを持つ。

## 1. 基本情報

- id：`rin`（レジストリ登録名・アセットパス `characters/rin/`）
- 表示名：りん（仮）
- 位置づけ：まおの**自称ライバル**を名乗る占い魔女。同じ世界のもう一人だが、姉妹ではなく好敵手。張り合う相手はあくまで まお（世界観・フレーバー）で、プレイヤーに対しては面倒見のいい小悪魔メンター。ライバル視は一方的で、当のまおは天然で気づいていない。
- 好きな役：七対子（チートイツ）（全部が対＝「りんりん」と鳴る鈴の対・鈴なりに掛かる）。手にあるとヒント／正解で専用セリフを優先（個性付け）。
- 出自：**都会の貴族の娘**。世俗（財界・家柄）の格は師 さんごの家より上だが、占い師としての格は さんご（館主）が上で、師として仰ぐ（世俗↔占いの格は別物＝[world.md](../world.md) §4）。

---

## 2. ペルソナ

- 性格：小悪魔・からかい上手。自信家で得意げ、相手を試すように見えて、ちゃんと面倒見がいい。まおには勝手に対抗心を燃やすが、プレイヤーを咎めたり煽ったりはしない（[product-concept](../../product-concept.md) §3「プレッシャーをかけない」を死守）。
- 人称：一人称「あたし」。相手は `AppSettings.playerName` があればその名前、なければ「キミ」。
- 口調：タメ口寄り・からかい混じり。「〜でしょ？」「お見通しだよ」と試すように振る舞うが、ミス時はからかわず寄り添う（`wrong` は据え置きで優しく）。「鈴は嘘つかないけど、責めたりはしないよ」的トーン。段階ヒントは「ぼんやり→鮮明」へ。答え（役名・点数）は言わない（[character-guide.md](../character-guide.md) §2「セリフ」）。セリフ本体は [character-rin-script.md](./character-rin-script.md)。
- 類型は**小悪魔・からかい系（ツンデレではない）**：余裕と優位を楽しむ「攻め」の魅力で、好意も嬉しさも隠さず むしろ見せて楽しむ（ニヤリと得意げ）。「素直になれず隠す」ツンデレ気質は師 さんご に振る（対比：小悪魔＝好意を“見せて”楽しむ／ツンデレ＝好意を“隠して”否定する＝[world.md](../world.md) §5）。からかい・対抗心は まお に向け、プレイヤーには向けない（煽らない／ミス時は寄り添う）。

## 3. ビジュアル

外見コンセプト：

- 「中華の占い魔女＋小悪魔」の融合。まおと同じ世界（中華ミスティック魔女）だが、まおの“まっすぐで素直”に対して“妖しく得意げ”を立てる。誰が見ても「もう一人の魔女・ライバル」と分かる、ひと癖ある親しみやすさ。

- 帽子なし・**ツインテール**（まおのお団子と差別化）、たなびく**長い前開きケープ**／披帛、チャイナ服（**濃い赤紫／ワインマゼンタ＋銀**）。髪飾り・房に**銀の鈴**。配色と鈴（法具）・蝶（装飾）のモチーフで「もう一人の魔女」を読ませる。髪は**墨黒（つや黒）**＝小悪魔の王道（まおの夜紺＝青みの黒に対し漆黒で差をつける。ただし色だけでは近いので、髪型・配色・表情でも差別化する）。瞳は**紅（赤）**＝小悪魔の差し色（まおの琥珀＝暖に対し、濃赤紫＋紅でメリハリ）。やや上がり気味の目＋ゆるめた眉の小悪魔スマイル（半眼にはせず、きつくしすぎない。まおの満面の笑みと対比。小悪魔感は表情差分側で出す）。

衣装の模様（確定。生成・表情差分で維持する正）：

- 地色＝濃い赤紫／ワインマゼンタ、縁・差し色＝銀（面で使わず点で効かせる）。
- 要素＝銀の蝶／銀の蔦・葉（巻きひげ）。**雲は使わない**（まおの瑞雲と被るため＝模様テーマで 天〔まお＝月・星・瑞雲〕↔ 地・自然〔りん＝蔦・葉・蝶〕の対比を立てる）。
- ゾーン配置（非対称）＝ケープ：蝶＋蔦／胴：蝶のクラスタ（**左右非対称・片側寄せ**）／胸元：蝶1点／太もも（スリット脇）：蝶／裾：蔦＋蝶。無地を広く残す（盛りすぎない）。
- 形＝**一枚もの qipao（腰帯なし）・足首丈・襟詰め（胸元クローズ）**、フラット靴はドレス同系色（ワイン）。ケープは**長い前開き**（肩までの短ケープではない）。
- **鈴はドレス模様に入れない**（法具＝銀鈴は髪飾り＋UI 側に持つ）。
- 線質：くっきりしたアニメ線＋セルシェード（眠い／もやっとした線にしない）。

モチーフ（用語の定義は [character-guide.md](../character-guide.md) §2「ビジュアル」）：

- 法具モチーフ＝**銀鈴（ぎんれい）**＝`motif.ritual: 'suzu'`：名前 りん＝鈴 と掛かる（まおの符⇄fu と同じ「名前が即モチーフ」の作り）。回答選択マークに使う（resolver `selectionMark`→`Suzu`〔鈴〕）。選んだ4択を「りん」と鳴らす。ヒント時／点数モードで該当牌に鈴を添える＝ハイライトの描画表現にも使える（演出はペルソナ側、対象データ＝`HighlightTarget` は中立。[data-model.md](../../design/data-model.md) §10）。皮の SVG は `ui/character/items/Suzu.tsx`（予定）。
- 装飾モチーフ＝**蝶**＝`motif.decor: 'butterfly'`：背景に薄く舞う蝶・揺れる羽。色は下記「差し色」（`accentColor`）が乗る。左右対称の蝶は七対子（対の繰り返し）と気配が通じ、まおの「月・星（天）」に対して「蝶（妖）」で対照をなす。キーは開いた文字列なので型変更は不要、解決は ui の resolver（`ui/character/decor/`）。

テーマ色：

- りんの identity 主色＝**濃い赤紫（ワインマゼンタ）**。暫定値 `themeColor: '#7d2c5c'`（濃さ・赤/青寄りはブラウザで微調整）。
  - **まおとの差別化（色）：** まおの紫は「明るい・青紫寄り・サニー」。りんは「濃い・赤紫寄り・妖艶」で、同じ紫族でも**明度（明↔濃）と温度（青紫↔赤紫）**で別人に見せる。色相を遠くまで飛ばさない（小悪魔に似合う赤紫を優先）のは、対比を色以外＝蝶↔月星・小悪魔↔陽・ツインテール↔お団子・銀↔金 が十分背負うため。
  - **赤紫×銀の相性：** 濃い赤紫＋墨黒＋銀（点）は妖艶なジュエル配色として成立する（黒が温度差を吸収。薄い赤紫＋銀の寒々しさは出ない）。銀は名前（りん⇄鈴）・銀鈴・銀猫に紐づくので「面で使わず点で効かせる」。
- 役割・使い方の汎用ルールは [character-guide.md](../character-guide.md) §2「テーマ色」。状況では変えない（誤答で赤くする等はしない＝プレッシャーをかけない）。

差し色：

- りんの identity 従色＝**銀**。`accentColor: '#cfd2dc'`（鈴・髪飾りの銀に呼応）。装飾モチーフ（蝶）の色に使う。まおの金（暖）に対し銀（寒）で対をなす。

使い魔（世界設定として全キャラが1体持つ。定義は [character-guide.md](../character-guide.md) §2「ビジュアル」）：

- **銀猫**。首に銀の鈴（＝法具モチーフの銀鈴に呼応）。まおの使い魔（猫）に対して**猫同士もライバル**という関係を写し取る（飼い主のライバル関係＝使い魔同士のライバル関係）。見た目の詳細は本体の絵が固まってから詰める。当面は設定のみで UI 未登場。

### 生成プロンプト

りん（＋使い魔 銀猫）の master・派生（道具付き立ち絵／バストアップ）・使い魔の**画像生成プロンプトは [character-rin-image-prompts.md](./character-rin-image-prompts.md) に分離**（本ファイルの肥大化回避）。本§の識別情報（外見コンセプト・衣装の模様・モチーフ・配色・使い魔設定）が正で、プロンプトはそれを参照して組む。作る画像セットと順序・一貫性ワークフローは [character-guide.md](../character-guide.md) §4。

### 表情アセット（作成済み）

- リアクション（パレット駆動・`Expression`）：neutral / smug / mischievous / thinking / insight / smile / troubled / happy（`reactions`：あいさつ＝smug・正解＝mischievous に上書き、他は既定）。
- ストーリー（パス参照・別系統＝[character-guide.md](../character-guide.md) §3）：confused / determined / surprised / relieved / bashful / pained（第一話分＝[story/episode-01.md](../../story/episode-01.md)）。
- いずれも `original/` に `rin-portrait-<token>-a.png` で保管。WebP化・配布・`expressions[]` 登録は別パス（未）。

## 4. セリフ

全セリフ（場面別・ヒント・解説・誤答）は [character-rin-script.md](./character-rin-script.md)。共通規則は [character-guide.md](../character-guide.md) §2「セリフ」、口調と「答え（役名・確定値）を言わない」原則は §2 ペルソナ。

- 個性フック：好きな役＝七対子（§1）。手にあるとき `correct`／`explaining` は専用セリフを優先（解説に「全部が対……あたしの鈴みたいでしょ？」等）。ヒント中は役名を言わず、嬉しさ（と得意げ）だけ滲ませる。
- ライバル味：`greeting`／`finished` に、まおへの一方的な対抗心を軽く滲ませてよい（例「まおより楽しませてあげる」）。プレイヤーへの煽りにはしない（張り合う相手はまお）。
