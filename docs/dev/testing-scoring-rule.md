# 採点ルールのテスト（必須ケース）

テストの全体方針は [testing.md](./testing.md)。本書は採点（役・符・点数）の必須ケース＝回帰スイートとして固定するもの。値の正は [scoring-rules.md](../spec/scoring-rules.md)。学習アプリで誤った点数を教えるのは致命的なので、ここを最優先で網羅する。

各ケースは `Hand`＋`Table`＋`WinContext`＋`RuleSettings`（= `score()` の入力一式）→ 期待 `{ totalHan, totalFu, scoreText, payments, rank, yaku一覧 }`。データ駆動・回帰の進め方は [testing.md](./testing.md) の「テストデータの出所と回帰方針」。

## 1. 符の特例

- 平和ツモ 20符固定（ツモ符を付けない）。
- 平和ロン 30符。
- 七対子 25符固定（切り上げしない）。
- 喰いピンフ形 → 30符繰り上げ。

## 2. 待ち符・面子符

- 単騎/嵌張/辺張 +2、両面/双碰 +0。
- 中張 明刻2/暗刻4、么九 明刻4/暗刻8。
- 槓子：中張 明8/暗16、么九 明16/暗32。
- 10符単位の切り上げ（端数→切り上げ）。

## 3. 雀頭符

- 役牌雀頭 +2。
- 連風牌雀頭：4符固定（場風＋自風で 2＋2）。

## 4. 点数・配分・境界

- 満貫境界：5翻 / 4翻40符 / 3翻70符（満貫）。その直下（4翻30符・3翻60符）が通常計算になること。
- 切り上げ満貫トグル：オフ時 4翻30符=子7700/親11600 等、オン時=満貫、の差分を両方テスト。
- 親子ツモ配分：子ツモ（親払い=子払い×2）、親ツモ（オール）。100点切り上げ。
- 跳満/倍満/三倍満の固定値と配分。
- 数え役満トグル：13翻が `kazoeYakuman:false` で三倍満、`true` で役満になる差分。

## 5. 役満

- 単一役満＝子32000/親48000。
- 四暗刻：ロンは不成立→対々和等に落ちる／単騎ロンは成立、の差分。
- 国士13面・大四喜 等の「ダブル扱い」が `doubleYakuman` トグルで切り替わること（オフ=単一役満どまり）。

## 6. 高点法

- 順子/刻子の両解釈が可能な手で、高得点の解釈が採用されること。
- 一盃口→二盃口、上位役への吸収（成立時に下位役を二重計上しない）。
