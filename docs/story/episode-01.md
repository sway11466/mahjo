# 第一話：市場の異変（仮題）

> ストーリーモード第一話（構想）。あらすじ・台本の正は本書。世界設定は [world.md](../characters/world.md)、ストーリーモード全体の枠は [backlog](../backlog.md)。
> 台本形式：`キャラ〔表情token〕：「セリフ」`。状況・動作は丸かっこのト書き。表情 token は各キャラの表情アセット名（[character-guide.md](../characters/character-guide.md) §3）。未制作キャラの表情は `〔—〕`。

## 登場人物

- まお（既定キャラ・見習い／素直・まっすぐ・人当たりよし）
- りん（見習い・自称ライバル／小悪魔・からかい）
- ひすい（姉弟子格・両世代の橋渡し／知的・聡明）※表情アセット未制作（キャラ立ち上げが前提）
- 漁師のおっちゃん（鯵を売る市場の顔なじみ。気は荒いが情に厚い。のち瘴気に呑まれて豹変）※NPCアート未制作
- 香辛料屋のおばちゃん（鯵の調理用の香辛料を売る。りんと遭遇する場の店番）※NPCアート未制作

## あらすじ

夕暮れの市場。まおが漁師のおっちゃんから鯵を買い、おまけをもらう。調理用の香辛料を買いに別の区画の香辛料屋へ行くと、りんと出くわす。そこへ魚屋の方向から悲鳴――駆け戻ると、さっきのおっちゃんが瘴気（しょうき）に呑まれて暴れている。その場に占い師は二人だけ。まおとりんが協力して祓う（＝ゲーム本編）。成功すればおっちゃんは正気に戻り、駆けつけたひすいが二人を褒める。失敗すれば二人にも瘴気が及び、ひすいが救済する（やり直し）。

## 本編

### シーン1：魚河岸

（夕暮れ。赤提灯が灯りはじめた市場。まおが漁師のおっちゃんの店先で鯵を選んでいる。）

漁師のおっちゃん〔—〕：「まおちゃん、今日は鯵がいいよ。ほら、一匹おまけだ」
まお〔smile〕：「わっ、ありがとうございます！ いつも助かります」
漁師のおっちゃん〔—〕：「いいってことよ。……その鯵、塩焼きもいいが、香辛料を効かせるとぐっとうまくなるぞ」
まお〔smile〕：「じゃあ、ちょっと香辛料を見てきますね。ごちそうさまです！」

### シーン2：香辛料屋

（市場の別の区画。香辛料の香りが漂う店先。おばちゃんが店番をしている。）

おばちゃん〔—〕：「あら、まおちゃん。今日は何にするんだい？」
まお〔neutral〕：「鯵に合う香辛料を……えーっと」

（後ろから、からかうような声がかかる。）

りん〔smug〕：「ふふっ、相変わらず市場の人気者ね、まお」
まお〔neutral〕：「あ、りん。買い出し？」
りん〔mischievous〕：「ええ。……おまけまでもらってさ。愛想がいいのも考えものね？」
まお〔troubled〕：「そ、そうかな……」
おばちゃん〔—〕：「ふふ、仲がいいねえ」

### シーン3：悲鳴

（そのとき、来た方向――魚河岸のあたり――から悲鳴があがる。）

まお〔surprised〕：「えっ……今の、悲鳴？」
りん〔surprised〕：「……っ。あっちよ、魚屋の方。急ぐわよ」
おばちゃん〔—〕：「な、なんだい騒がしい……あんたたち、気をつけるんだよ！」

### シーン4：瘴気

（駆け戻る二人。さっきの漁師のおっちゃんが、包丁を振り回して暴れている。その体には、占い師にしか視えない黒い瘴気がまとわりついている。）

まお〔confused〕：「おっちゃん……！？ さ、さっきまであんなに優しかったのに……ど、どうしよう……っ」
りん〔confused〕：「あれは……瘴気。まさか、こんな町中で……」

（周囲を見回すが、占い師らしき大人の姿はない。）

りん〔confused〕：「視えてるのも、祓えるのも……今ここには、あたしたちだけよ」

### シーン5：覚悟

まお〔determined〕：「……うん。わたしたちが、祓うしかない。おっちゃんを、助けたい」
りん〔smug〕：「ふ……まおにしては、いい目をするじゃない」
りん〔determined〕：「いいわ。あたしの読み、見せてあげる。――行くわよ」

（＝ここからゲーム本編。瘴気を祓う＝和了を決める。ルール詳細は backlog ストーリーモード参照・TBD。）

## 成功ルート

（見事に祓い、黒い瘴気が晴れる。漁師のおっちゃんがへたり込み、我に返る。）

漁師のおっちゃん〔—〕：「……あれ、おれは一体……？ なんだか、悪い夢を見ていたような……」
まお〔relieved〕：「よかった……！ おっちゃん、元に戻った……」
りん〔relieved〕：「ふぅ……まったく、肝が冷えたわ」

（騒ぎを聞きつけ、ひすいが駆けつける。）

ひすい〔—〕：「無事みたいね。……この瘴気を、あなたたち二人で祓ったの？ えらいわ、よくやったわね」
まお〔happy〕：「ひすいさん！ えへへ、二人でがんばりました！」
りん〔bashful〕：「べ、別に……当然のことをしただけよ」

（ひすいが、晴れたばかりの空を見上げる。）

ひすい〔—〕：「……それにしても、こんな町中で瘴気が出てくるのは珍しい。何かが起こる予兆でなければ、いいのだけれど……」

## 失敗ルート

（祓いきれず、瘴気が二人にも及ぶ。肉体の苦痛だけでなく、負の感情が噴き出してくる＝瘴気の症状・[world.md](../characters/world.md) §3。）

まお〔pained〕：「う……っ、頭の中に、いやな気持ちが、押し寄せて……っ」
りん〔pained〕：「くっ……これが、瘴気……っ。しっかりしなさい、まお……！」

（そこへ、ひすいが現れる。）

ひすい〔—〕：「――そこまで。あとは、わたしが」

（ひすいが得意役〔未決〕で一気に瘴気を祓い、二人を解放する。）

ひすい〔—〕：「無茶をして。……でも、よく踏みとどまったわね。続きは、もっと腕を上げてからよ」

（※失敗はクリア扱いにならず、やり直し。一度見たストーリーはスキップ可。）

## 必要表情・アセット対応

- まお：smile / neutral / troubled / surprised / confused / determined / relieved / happy / pained（すべて作成済み）
- りん：smug / mischievous / surprised / confused / determined / relieved / bashful / pained（すべて作成済み）
- ひすい：表情アセット未制作（褒める笑顔・案じる顔・祓う決意 が必要。キャラ立ち上げが前提）
- 漁師のおっちゃん：NPCアート未制作（通常／瘴気で豹変・暴れる／正気に戻る）
- 香辛料屋のおばちゃん：NPCアート未制作（店番・遭遇の場）
