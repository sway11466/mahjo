設定値の説明（意味・取り得る値）

主旋律

楽器 instrument音色。撥弦(琴) / 鐘 / 木琴 / 笛 / 弓 の5種
主音 tonic曲の中心の音＝キー。C〜B ＋ オクターブ 3〜6（テキストは C5 の形）
モード mode音階の色合い。宮 / 商 / 角 / 徵 / 羽（宮＝明るい 〜 羽＝暗い）
テンポ tempo速さ。40〜120 BPM
明るさ brightnessこもり↔きらびやか（ローパス遮断周波数）。800〜7000 Hz
余韻 sustain歯切れ↔伸びる。0〜1
melody度数記法：度数 1〜5 ／ '上 ,下 ／ :拍（既定1）／ 0休符 ／ 空白区切り

即興音（セグメントの各項目）

start開始小節（0＝先頭）。同じ start に複数セグメントを置くと重なって同時に鳴る（声部を重ねられる）
length続く小節数。start＋length が主旋律の小節数を超えないように
instrument楽器（5種）。主旋律と別に選べる
densityその小節で鳴らす音数。0＝休み。目安 0〜6
octave主旋律の主音から何オクターブ上か。0＝同じ高さ ／ 1＝1つ上 ／ 2＝2つ上（目安 0〜3・負も可）
offset拍からのずらし（拍数）。0＝拍ぴったり ／ 0.5＝裏拍。音は8分グリッドに整列
volume音量。0〜1（主旋律は約0.6）
brightness明るさ（この即興音のローパス）。800〜7000 Hz
sustain余韻。0〜1
① 主旋律（パラメーターUI・キー/モード/テンポは即興音と共有）
② 主旋律（テキスト・直接編集可／UIと同期）
③ 即興音（セグメント配列＝length＋特性／小節グリッドで確認）
④ 再生（主旋律＋即興音を重ねる）
